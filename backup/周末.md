*—— 那些被我拖延的，不止是时间，还有未完成的自己*

---

### 一、空白的开场

周末是两段被拉长的空白，像一张没来得及落笔的纸，摊在眼前，却让人手足无措。

明明是从紧绷里抽离出来的时间，却常常在醒来的那一刻就开始发虚：

> 窗外很安静，手机也安静，没有必须赶的截止日期，没有催进度的消息弹窗，世界好像暂时把我放下了，我却不知道该把自己放在哪儿。
> 
> 

这种空虚，像浸了水的棉絮，轻飘飘却压得人提不起劲，连阳光落在窗台，都显得有些多余。

### 二、搁浅的计划

每个周五的晚上，我总会在备忘录里列下一串「周末清单」，加粗的字迹里满是期待，却从来没真正完成过：

- **手工**：把搁置了半个月的羊毛毡材料翻出来，完成那只没做完的小猫（材料包上的灰尘，已经厚到能看清指纹）

- **博客更新**：把攒了一周的碎碎念整理成文，配上上周拍的风景图，标题都想好了——《周末的空白与慌张》

- **软件开发**：补全那个简易备忘录软件的「备份功能」，代码停在第128行，已经停了三个周末

这些计划，像被潮水冲上沙滩的贝壳，看着鲜活，却被我一次次搁置在原地。

我总安慰自己：「周末就是用来休息的，不用逼自己太紧」，可这份「宽容」，最后都变成了深夜的懊悔。

周六的上午，往往是在「再赖5分钟」的自我妥协里溜走的。等真正坐起身，已是正午，随便煮一碗面，刷着无关紧要的短视频，不知不觉就到了下午。

试着打开电脑：

```plain text
// 打开博客编辑页，光标闪烁了十分钟
// 点开软件开发工具，看着未完成的代码，大脑一片空白
// 找出手工材料，摆弄了两下毛线，又默默放回盒子里
```

什么都想做，什么都做不下去，时间就在这种纠结里，悄悄从指缝间溜走。
